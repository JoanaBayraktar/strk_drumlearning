# STRK

STRK is a browser-based drum practice prototype inspired by rhythm games and drum notation players.

## Run locally

Serve this folder with any static web server, for example:

```sh
python3 -m http.server 8765
```

Then open:

```text
http://127.0.0.1:8765/
```

## Included prototype features

- Song library start screen
- Horizontal drum-note player
- Keyboard hit detection with score, combo and accuracy
- Adjustable speed, zoom, hit range and backing volume
- Loop practice controls
- Local MIDI imports rendered as drum charts
- Sample-based drum, bass and guitar playback
